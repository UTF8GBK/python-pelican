快速上手

创建您的GitHub仓库

1、克隆仓库

    git clone https://github.com/your-username/your-repository.git

2、创建Python虚拟环境

Linux and Mac

  • 进入项目：cd <项目>
  • 创建虚拟环境：python3 -m venv pelican_venv
  • 激活虚拟环境：source pelican_venv/bin/activate

Windows

  • 打开命令提示符（Windows键+R，然后输入cmd，按Enter）或者 Powershell。
  • 安装 virtualenv：pip install virtualenv
  • 创建 Python 虚拟环境: virtualenv -p C:/Python311/python.exe py36env
  • 激活虚拟环境：cd py36env/Scripts , activate

3、配置 pelican

  • 安装 pelican pip install "pelican[markdown]"

  • 生成 pelican pelican-quickstart 配置如下：

    • Where do you want to create your new web site? [.] 【网站路径，默认即可】

    • What will be the title of this web site? 随意写【网站名称】

    • Who will be the author of this web site? 【作者】

    • What will be the default language of this web site? [en] 【语言】

    • Do you want to specify a URL prefix? e.g., https://example.com (Y/n) n

    • Do you want to enable article pagination? (Y/n) n 【启用文章分页】

    • What is your time zone? [Europe/Rome] 【默认】

    • Do you want to generate a tasks.py/Makefile to automate generation and publishing? (Y/n) y 【生成一个tasks.py/Makefile来自动生成和发布】

    • Do you want to upload your website using FTP? (y/N) n 【使用 FTP 上传你的网站】

    • Do you want to upload your website using SSH? (y/N) n 【 SSH 上传你的网站】

    • Do you want to upload your website using Dropbox? (y/N) n 【使用 Dropbox 上传你的网站】

    • Do you want to upload your website using S3? (y/N) n 【使用 S3 上传你的网站】

    • Do you want to upload your website using Rackspace Cloud Files? (y/N) n 【使用 Rackspace Cloud Files 上传你的网站】

    • Do you want to upload your website using GitHub Pages? (y/N) n 【使用 Github pages 上传你的网站】

  • 启动 pelican：

    • 在启动之前先在content创建一个images的图片
    • 在content 创建第一篇md文章
    • 在根目录运行 pelican content
    • 查看运行记录 pelican -l 本地会生成一个http://127.0.0.1:8000 ，打开即可获取您的网站
